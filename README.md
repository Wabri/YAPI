# YAPI - Yet Another Package Installer
This Package Installer only works on Debian currently.

**_Work in Progress_**..
Current Packages Supported:
 -  Atom
 -  Discord
 -  Dropbox
 -  Eclipse
 -  Gitkraken
 -  Google Chrome
 -  Java jdk 11 (Includes home path)
 -  Light
 -  Node JS
 -  Nvidia Drivers
 -  Oh My Zsh
 -  Playerctl
 -  Rambox
 -  Skype
 -  Steam
 -  VirtualBox
